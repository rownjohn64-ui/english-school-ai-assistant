# 🎓 AI-ассистент онлайн-школы английского языка

Мультимодальный Telegram-бот на Python, который помогает ученикам изучать английский язык, отвечает по базе знаний школы, понимает голосовые сообщения и анализирует изображения с заданиями.

---

## ✨ Возможности

- 💬 обычный текстовый режим;
- 📚 RAG-ответы по документам из папки `data/`;
- 🎙️ распознавание голосовых сообщений;
- 🔊 генерация голосового ответа;
- 🖼️ анализ изображений и учебных заданий;
- 🧠 хранение истории диалога;
- 📄 загрузка и индексация документов;
- 📊 просмотр состояния базы знаний;
- 🔄 переключение режимов работы.

---

## 🧩 Режимы работы

| Режим | Команда | Назначение |
|---|---|---|
| Text | `/mode text` | Общие вопросы по английскому языку |
| RAG | `/mode rag` | Ответы по документам онлайн-школы |
| Voice | `/mode voice` | Работа с голосовыми сообщениями |
| Reset | `/reset` | Очистка истории диалога |

Дополнительные команды:

```text
/start  — запуск бота
/help   — справка
/index  — индексация документов из папки data/
/stats  — состояние базы знаний
```

Изображения можно отправлять без отдельной команды — бот автоматически распознаёт их и анализирует содержимое.

## 🚀 Установка и запуск

### 1. Перейдите в папку проекта

```powershell
cd C:\PythonProject\assistent_study
```

### 2. Создайте виртуальное окружение

```powershell
python -m venv venv
```

### 3. Активируйте окружение

```powershell
venv\Scripts\Activate.ps1
```

После активации в терминале появится:

```text
(venv)
```

### 4. Установите зависимости

```powershell
pip install -r requirements.txt
```

### 5. Создайте файл `.env`

Скопируйте `.env.example` в `.env` и заполните переменные:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
OPENAI_API_KEY=ваш_api_ключ
OPENAI_BASE_URL=https://api.openai.com/v1

TEXT_MODEL=gpt-4o-mini
VISION_MODEL=gpt-4o-mini
STT_MODEL=whisper-1
TTS_MODEL=gpt-4o-mini-tts
TTS_VOICE=alloy
EMBEDDING_MODEL=text-embedding-3-small

ADMIN_ID=0
```

> Никогда не публикуйте файл `.env` и не добавляйте его в открытый репозиторий.

### 6. Запустите бота

```powershell
python main.py
```

## 📚 Подготовка базы знаний

1. Поместите документы в папку `data/`.
2. Перезапустите бота.
3. Выполните команду `/index`.
4. Проверьте состояние базы командой `/stats`.

Пример результата:

```text
Статус базы знаний:
• Коллекция: english_school_docs
• Документов: 3
• Чанков: 3
• Статус: индексация завершена
```

## 🧪 Примеры использования

### Обычный текстовый режим

```text
/mode text
```

Пример запроса:

```text
Объясни разницу между do и does.
```

### RAG-режим

```text
/mode rag
```

Пример запроса:

```text
За сколько часов нужно предупредить о переносе занятия?
```

Пример ответа:

```text
Ученик должен предупредить о переносе занятия не позднее чем за 12 часов.

Источник: Правила_обучения.txt
```

### Голосовой режим

```text
/mode voice
```

После этого отправьте голосовое сообщение. Бот распознает речь и сформирует ответ.

### Анализ изображения

Отправьте фотографию упражнения по английскому языку. Бот:

- распознает текст;
- определит тему;
- объяснит задание;
- предложит дать ответы, проверить ответы или разобрать правило.
